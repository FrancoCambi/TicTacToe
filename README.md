# 🎮 TicTacToe

A simple TicTacToe game built with Python and Pygame.

---

## ✨ Features

- 🎲 Two game modes:
  - 👤 vs 👤 **Player vs Player**
  - 🧠 vs 👤 **Player vs AI**
- 🖱️ Interactive graphical interface using Pygame
- ⚡ Easy to run and play

---

## 📦 Requirements

- Python 3.x
- Pygame

Install Pygame with:

```bash
pip install pygame
```

---

## ▶️ How to Run

### 🤖 Player vs AI

```bash
python TicTacToe_VS_AI/main.py
```

### 👥 Player vs Player

```bash
python TicTacToe_VS_PLAYER/main.py
```
